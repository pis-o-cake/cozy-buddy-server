# Cozy Buddy — 기능 명세서 (Nest Hub 2세대 벤치마크)

> Google Nest Hub 2세대(2021, 7" 스마트 디스플레이)를 벤치마크해 cozy-buddy가 구현할 기능을 정의.
> **핵심 전략**: Nest Hub의 약점(딸리는 LLM, 클라우드·구독 종속)을 **강한 로컬 LLM + 로컬 우선 + 무료**로 뒤집는다.

---

## 범례

| 표기 | 의미 |
|---|---|
| **우선순위** | 🟢 P0(MVP) · 🟡 P1(1차 확장) · 🔵 P2(후순위) · ⚫ 제외 |
| **처리** | `로컬` 온디바이스/집안 · `클라우드` 외부 API · `하이브리드` |
| **HW** | ⚠️ 특수 하드웨어 필요 (기본 태블릿엔 없음) |

| 포토 프레임 | Google Photos 슬라이드쇼 | 로컬 폴더/NAS 사진 슬라이드쇼 | 🟡 | 로컬 |

## 2. 음성 비서 (핵심 대체 타깃)

| 기능 | Nest Hub | cozy-buddy | 우선순위 | 처리 |
|---|---|---|---|---|
| 웨이크워드 | "Hey Google" | openWakeWord 커스텀 호출어 | 🟢 | 로컬 |
| 명령/Q&A | 온디바이스 일부 + 클라우드 | **로컬 LLM(강점)** / 클라우드 옵션 | 🟢 | 하이브리드 |

| **화자 구분(Voice Match)** | 최대 6명 개인화 | 🔵 v1은 단일 가구, `user_id`만 열어둠 | 🔵 | 로컬 |
| 통역 모드 | 실시간 양방향 ~29개국어 | 🔵 클라우드 STT/번역/TTS 조합 시 가능 | 🔵 | 클라우드 |
| 다국어 | 다수 언어 | 한국어 우선, i18n 리소스 분리 | 🟢 | — |

> **차별점**: **LLM 품질이 이 프로젝트의 존재 이유.** Nest Hub의 제일 큰 약점을 정면 공략.

## 3. 수면 감지 (Nest Hub 최대 차별점 — 최대 난관)

| 기능 | Nest Hub | cozy-buddy | 우선순위 | 처리 |
|---|---|---|---|---|
| 움직임/호흡 감지 | Soli 60GHz 레이더 | ⚠️ **Soli 단독 판매 안 됨** → mmWave 프레즌스 모듈 or 침대센서 or BLE/WiFi CSI | 🔵 | 로컬 |
| 기침/코골이 감지 | 온디바이스 오디오 분류 | 온디바이스 오디오 분류기로 **대체 가능** | 🔵 | 로컬 |
| 수면 지표(효율·호흡수) | 라디오 계산 | 센서 확보 시 산출 | 🔵 | 로컬 |
| 온도/조도 방해요소 로깅 | 내장 센서 | 태블릿 센서 or 외부 IoT 센서 | 🔵 | 로컬 |
| 코칭/리포트 | Wellness 페이지 | LLM 기반 리포트로 **차별화 가능** | 🔵 | 하이브리드 |
| **구독** | Fitbit Premium 예정(~$10/월) | **무료 제공 = 강력한 차별점** | — | — |

> **판단**: 하드웨어 의존이 커서 **v1 스코프 제외, P2 실험**. 단, 구현 시 "무료 + LLM 코칭"으로 Nest Hub 대비 우위.

## 4. 스마트홈 제어 (cozy-buddy 본진)

| 기능 | Nest Hub | cozy-buddy | 우선순위 | 처리 |
|---|---|---|---|---|
| 홈 대시보드 | Home View 방별 그룹 | 기기 카드(방별) 터치 제어(v2 §10) | 🟢 | 로컬 |
| 음성 제어 | "불 켜줘" | LLM Tool Calling → device 도메인 | 🟢 | 하이브리드 |
| 카메라 피드 | 화면에 라이브 | RTSP/WebRTC 로컬 스트림 | 🟡 | 로컬 |
| 카메라 이벤트 이력 | Nest Aware 구독 | **로컬 저장 = 무료 차별점** | 🔵 | 로컬 |
| 루틴/자동화 | Google Home 루틴 | scenario 도메인 + 스케줄러(v2) | 🟢 | 로컬 |
| **Matter** | 컨트롤러 지원 | 🔵 후순위, 어댑터 추가로 확장 | 🔵 | 하이브리드 |
| **Thread 보더라우터** | 내장 802.15.4 | ⚠️ 태블릿에 없음 → **Thread 동글/별도 허브** | 🔵 | 로컬 |

> **차별점**: 음성 제어의 자연어 이해력이 LLM 덕에 우월. 카메라 이력 로컬 무료.

## 11. MVP 스코프 (Nest Hub 대비)

**🟢 P0 (반드시)** — 이것만으로 "Nest Hub + 똑똑한 LLM" 성립
- 온디바이스 웨이크워드 → **로컬 LLM 음성비서**(핵심 차별점)
- 스마트홈 음성/터치 제어 + 방별 대시보드
- 앰비언트(시계·자동밝기·디밍) + 카드 UI
- 타이머/알람 + Good Morning/Night 루틴

**🟡 P1** — 완성도
- 포토프레임, 인터콤(LAN 방송), 카메라 피드, BT 오디오, 날씨/뉴스, 리마인더, 선라이즈 알람, 마이크 뮤트 토글

**🔵 P2** — 실험/확장
- 수면감지(HW 확보 시, **무료+LLM코칭**), Matter/Thread, 멀티룸 오디오, 통역, 화자구분, 영상통화(카메라 추가)

**⚫ 제외**
- Soli 에어제스처(정통), 큐레이션 아트, 음성통화
